## Prova obrigatória de Sistemas Distribuídos

**Criar um WebService (API)** - *GRAPHQL* 
 

- Lista alunos; 
- Pesquisa um aluno específico; 
- Cadastrar um aluno;
- Editar um aluno;
- Remover um aluno.

**A entidade aluno é:** 

- Nome (String);
- Curso (String);
- Semestre (Int);
- RA (Int);
- CPF (Int);
- Cidade (String).
